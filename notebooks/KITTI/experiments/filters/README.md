## Description

This repository contains notebooks checking each filter's behavior for each setup including (IMU, VO), (IMU, VO+GPS), (INS, VO+GPS). 

The dataset utilized in the notebook is KITTI raw dataset sequence number 09.

## Note
Please download KITTI dataset before running the notebook.

To download the KITTI dataset, navigate to the root directory and run `./kitti_data_downloader.sh`.
