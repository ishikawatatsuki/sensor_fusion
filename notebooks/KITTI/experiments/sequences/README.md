## Description
This repository contains notebooks that apply all implemented filters to KITTI raw dataset sequence number 04 and 09.
The sequence number is suggested by the filename:
* 04-test.ipynb uses sequence number 04
* 09-test.ipynb uses sequence number 09

## Note
Please download KITTI dataset before running the notebook.

To download the KITTI dataset, navigate to the root directory and run `./kitti_data_downloader.sh`.
